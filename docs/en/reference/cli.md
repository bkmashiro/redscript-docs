# CLI Reference

The RedScript command-line interface for compiling `.mcrs` files to datapacks.

## Installation

```bash
npm install -g redscript-mc
```

## Commands

### redscript compile

Compile a RedScript file to a Minecraft datapack.

```bash
redscript compile <input> [options]
```

**Arguments:**

| Argument | Description |
|----------|-------------|
| `<input>` | Path to `.mcrs` file or directory |

**Options:**

| Option | Description | Default |
|--------|-------------|---------|
| `-o, --output <path>` | Output directory | `./out` |
| `-n, --namespace <name>` | Datapack namespace | File name |
| `--pack-format <version>` | Pack format version | Latest |
| `--description <text>` | Pack description | `"Generated by RedScript"` |
| `--minify` | Minimize output size | `false` |
| `--no-comments` | Strip comments from output | `false` |

**Examples:**

```bash
# Basic compilation
redscript compile hello.mcrs

# Specify output directory
redscript compile hello.mcrs -o ./my-datapack

# Set namespace and description
redscript compile game.mcrs -n minigame --description "My Mini Game"

# Minified output
redscript compile game.mcrs -o ./dist --minify
```

### redscript watch

Watch for file changes and recompile automatically.

```bash
redscript watch <input> [options]
```

Accepts the same options as `compile`.

```bash
redscript watch game.mcrs -o ./server/world/datapacks/my-game
```

### redscript init

Create a new RedScript project with boilerplate files.

```bash
redscript init [name]
```

```bash
redscript init my-datapack
```

Creates:
```
my-datapack/
├── main.mcrs
└── redscript.json
```

### redscript check

Type-check a file without producing output.

```bash
redscript check <input>
```

```bash
redscript check game.mcrs
```

### redscript version

Show version information.

```bash
redscript --version
```

## Configuration File

`redscript.json` — optional project configuration:

```json
{
  "namespace": "my_game",
  "description": "My Minecraft Mini Game",
  "packFormat": 48,
  "entry": "main.mcrs",
  "output": "./dist",
  "minify": false
}
```

| Field | Description |
|-------|-------------|
| `namespace` | Datapack namespace |
| `description` | Pack description |
| `packFormat` | Minecraft pack format version |
| `entry` | Entry file |
| `output` | Output directory |
| `minify` | Minimize output |

## Exit Codes

| Code | Meaning |
|------|---------|
| `0` | Success |
| `1` | Compilation error |
| `2` | File not found |
| `3` | Invalid configuration |
