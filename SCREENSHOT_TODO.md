## Screenshot Guidelines

1. Use a clean, default resource pack
2. Resolution: 1920x1080 or 1280x720
3. Hide HUD elements except the relevant UI (F1 to toggle HUD)
4. Use `/gamemode spectator` for clean backgrounds when needed
5. Save as PNG, place in `docs/en/tutorials/images/` with descriptive names

Example naming:
- `title-hello-world.png`
- `actionbar-timer.png`
- `effect-glowing.png`
